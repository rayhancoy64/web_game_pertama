Tentang Proyek Ini

selamat datang di SIMPELGAME! Website ini dibangun sebagai tugas Ujian Akhir Semester (UAS) mata kuliah Pemrograman web. Tujuan dari website ini adalah untuk portfolio dari game -game yang nantinya saya akan saya buat, agar memudahkan bagi siapa saja yang ingin mencoba game buatan saya.

Website ini dibangun menggunakan HTML, CSS, dan JS, mengimplementasikan fitur seperti Flexbox, CSS Grid, dan tata letak yang ramah pengguna.
